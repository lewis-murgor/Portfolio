Pitch Point

This is a web application that displays pitches posted by users. A user is required to sign up and login to post a pitch or upvote and downvote other pitches A user is also required to login to comment on other pitches posted. A user can post either a wise pitch or a pickup line or a motivational pitch. The web application was built using Python flask framework, HTML, CSS and postgresql as the database.

The web application was built using Python Django framework, HTML, CSS and postgresql as the database.

Akan Names

This is a web application that takes a user's birthday, calculates the day of the week the user was born and dislays the user's akan name according to the user's gender. Akan names are derived from the Ghanian culture where children are named according to the day of the week they were born. The web application was built using HTML, CSS and Javascript.


Insta

Insta is a web application that resembles instagram. It requires a user to sign up and login to start using the web application. It allows users to post photos and view photos posted by other users. It also allows a user to like photos and comment on photos. It was built using Python django, HTML, CSS and postgresql as the database.


Neighborhood

Neighborhood is a web application that shows neighborhoods and businesses in the neighborhood. It requires a user to sign up and login to create or join a neighborhood. A user can also add a business in the neighborhood and also post for other users to see the posts. It was built using Python django, HTML, CSS and postgresql as the database.
