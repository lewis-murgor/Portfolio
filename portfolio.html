Blog

This is a blog application that displays blogs posted by users. It requires a user to create an account and login to post a blog and comment on other blogs. The web application was built using Python flask framework, HTML, CSS and postgresql as the database


Pitch Point

This is a web application that displays pitches posted by users. A user is required to sign up and login to post a pitch or upvote and downvote other pitches A user is also required to login to comment on other pitches posted. A user can post either a wise pitch or a pickup line or a motivational pitch. The web application was built using Python flask framework, HTML, CSS and postgresql as the database.


Gallery

This is a web application that displays photos posted by users. Users can post photos and view photos posted by other users. They can also view details of a single photo by clicking on the photo. The web application was built using Python Django framework, HTML, CSS and postgresql as the database.
